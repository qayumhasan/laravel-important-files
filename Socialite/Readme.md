## Install Laravel 8
 ``` composer create-project --prefer-dist laravel/laravel facebookLogin ````

## Install JetStream
``` composer require laravel/jetstream ````
``` php artisan jetstream:install livewire ```
``` npm install ```
``` npm run dev ```
```` php artisan migrate ```
##  Install Socialite
 ``` composer require laravel/socialite ```
